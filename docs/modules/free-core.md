# Free Core Modules

These seven modules are part of the free core and work indefinitely without a license.

## Product Filters

AJAX-powered filtering for shop and category pages. Customers narrow products by price, attributes, categories, tags, ratings, and custom taxonomies — results update without a full page reload. See the [Product Filters guide](/guides/product-filters).

## Reviews

Extends WooCommerce reviews with richer review widgets and **verified-purchase** indicators so shoppers can trust the feedback they read. Works alongside the native WooCommerce review system. See [Reviews, Wishlist & Compare](/guides/reviews-wishlist-compare).

## Wishlist

Lets logged-in (and, optionally, guest) customers save products to a personal list they can return to later. A common conversion lever for stores with considered purchases.

## Compare List

Side-by-side comparison of products on a single screen — attributes, price, and key fields lined up so shoppers can decide faster.

## Social

- **Share buttons** on product and content pages.
- **Social meta tags** (Open Graph / Twitter Card) so shared links render rich previews.
- **Social sign-in** for faster account creation and checkout.

## SEO

Basic search-engine helpers: product schema (structured data), meta tags, and sitemap helpers. This is intentionally lightweight and coexists with dedicated SEO plugins; it covers the WooCommerce-specific gaps.

## Dynamic Pricing (basic)

Two everyday pricing rules with no license:

- **Quantity discounts** — buy more, pay less per unit.
- **Percentage-off** rules.

For tiered/role-based/scheduled pricing, see the premium [Dynamic Pricing guide](/guides/dynamic-pricing).

## Enabling free modules

All seven are available the moment you install and activate Empora. Toggle each on under **Empora → Modules** and configure it on its tab under **Empora → Settings**. No license, no checkout, no time limit.
