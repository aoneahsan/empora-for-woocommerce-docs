# Empora for WooCommerce

**Empora for WooCommerce** (formerly *All-In-One WooCommerce*) is a comprehensive toolkit that extends WooCommerce with conversion-focused storefront features and back-office automation. It ships with a freely usable core and a set of optional premium modules unlocked by a paid license.

A license is **only** required to enable premium modules. The free core stays fully functional indefinitely without one.

## What you get

- **Free core (no license required)** — Product Filters, Reviews, Wishlist, Compare List, Social sharing, basic SEO, and basic Dynamic Pricing.
- **Premium modules (license required)** — Advanced Shipping, Payments Hub, Taxes, Inventory, Bulk Edit, Import/Export, Reports, plus dozens of optional modules such as Subscriptions, Bookings, Gift Cards, Rewards Points, Multi-Currency, Email Automation, Shipment Tracking, and integrations (Square, Google Listings & Ads, Klaviyo).
- **One unified admin** — a React-powered admin panel with a Modules grid, a single Settings store, a Dashboard showing license and module status, and a Reports view.
- **HPOS-compatible** — declares full compatibility with WooCommerce High-Performance Order Storage.

## Who it is for

- Store owners who want a single, well-maintained plugin instead of stitching together ten single-purpose plugins.
- Developers who want a clean, modular codebase (each feature is an isolated module that can be enabled or disabled independently).
- Agencies that need a consistent toolkit across many client stores, with a license model that scales by number of sites.

## How the product fits together

Empora is a small ecosystem, not just a plugin:

| Surface | What it is |
|---|---|
| **WordPress plugin** | The store extension itself. Installs on any WooCommerce site. Contains the free core and all premium modules. |
| **Marketing + dashboard site** | [empora.aoneahsan.com](https://empora.aoneahsan.com) — buy a license, read docs, and manage your license + connected stores. |
| **Backend API** | The license + billing service the plugin and dashboard talk to (activation, validation, entitlements, Stripe checkout). |

You only ever install the **plugin**. The site and API are hosted services that the plugin and your account use behind the scenes.
